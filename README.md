# Battleship Game (Морской бой)

Консольная реализация классической игры “Морской бой”. Проект был полезен для отработки: структур данных, координатных сеток, правил игры и аккуратной логики валидации.

## Возможности
- Режим 1v1 (двое игроков за одним компьютером) 
- Расстановка кораблей с проверкой правил
- Ходы, попадания/промахи, уничтожение кораблей
- Отображение поля и “тумана войны”

## Технологии
- Java 
- Тесты: JUnit
